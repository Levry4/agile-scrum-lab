# agile-scrum-lab
Лабораторная работа 5 по Agile и Scrum
Простое веб-приложение для управления задачами. Можно добавлять задачи, отмечать их как выполненные и удалять. Задачи можно помечать тегами для удобной фильтрации.
Сделано на HTML, CSS и JavaScript. Работает в браузере.

## Sprint 1 Review

        Что сделали:
- Придумали название приложения
- Создали макет главного экрана
- Составили список функций

        Что не успели:
- Подготовить презентацию для стейкхолдеров

## Sprint 1 Retrospective

        Что было хорошо:
- Задачи были понятными
- Работали быстро

        Что было плохо:
- Неправильно оценили время на презентацию
- Мало общались в процессе

        Что улучшим:
1. Лучше оценивать время выполнения
2. Проводить ежедневные стендапы
3. Дробить крупные задачи
